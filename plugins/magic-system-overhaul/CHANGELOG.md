# Changelog - Magic System Overhaul Plugin

## [1.0.0] - 2026-01-16

### Added
- Initial release of Magic System Overhaul plugin
- 6 magic system modes (normal, unlimited MP, zero cost, HP-based, item-based, free magic)
- Unlimited MP mode (set all characters to 9999 MP)
- Zero MP cost mode (all spells cost 0 MP)
- MP cost multiplier (0x to 10x)
- Magic power multiplier (0.1x to 10x damage)
- Spell unlocking (unlock all 54 spells for characters)
- Alternative cost systems (HP-based, item-based - experimental)
- Preset configurations:
  - FF7 Materia-style (all spells, unlimited MP, normal costs)
  - MP-free system (unlimited + zero cost + all spells)
  - Low-MP challenge (2x costs)
  - High-MP bonus (0.5x costs)
- Magic system status display
- Configuration export
- Backup and restore system
- ~650 lines of code, ~6,800 words documentation

### Features
- **6 Modes:** Complete magic system overhauls
- **MP Control:** Unlimited, zero, or custom values
- **Cost Modification:** 0-10x multipliers
- **Spell Unlocking:** All 54 FF6 spells
- **Power Scaling:** 0.1-10x magic damage
- **Presets:** Quick-apply popular configurations

### Use Cases
- Challenge runs (zero MP, MP-free)
- Build testing (unlimited resources)
- Speedrunning (unlimited MP)
- Experimentation (alternative cost systems)
- Power fantasy (max everything)
