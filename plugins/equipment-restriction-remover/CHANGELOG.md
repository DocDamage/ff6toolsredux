# Changelog - Equipment Restriction Remover Plugin

## [1.0.0] - 2026-01-16

### Added
- Initial release of Equipment Restriction Remover plugin
- Character restriction removal (any character, any equipment)
- Gender restriction removal (male characters wear female items)
- Universal dual-wielding (all characters dual-wield)
- Relic stacking (same relic in both slots)
- Equipment conflict removal (conflicting relics compatible)
- Unrestricted mode (removes ALL restrictions at once)
- Force equip system (bypass validation)
- Unrestricted build application
- Restriction status display
- Stat calculation with unrestricted equipment
- Backup and restore system
- ~570 lines of code, ~6,000 words documentation

### Features
- **5 Restriction Types:** Character, gender, dual-wield, relic stacking, conflicts
- **Unrestricted Mode:** One-command removal of all restrictions
- **Force Equip:** Bypass validation for any equipment
- **Example Builds:** Double Economizer, Genji Glove + Offering, etc.

### Use Cases
- Extreme build experimentation (double Economizer, double Offering)
- Gender equality (male characters wear Minerva Bustier)
- Build testing (any equipment combination)
- Overpowered challenge runs
