# Changelog - No Level System Plugin

## [1.0.0] - 2026-01-16

### Added
- Initial release of No Level System plugin
- Fixed level mode (lock all characters at specific level 1-99)
- 4 flat stat presets (low/balanced/mid/high)
- Experience gain control (disable or multiply 0-10x)
- Combined no-level mode (fixed level + flat stats + no exp)
- Challenge configurations (low-level, equipment-only)
- Esper bonus application at level 1
- Backup and restore system
- Operation logging
- ~520 lines of code, ~5,500 words documentation

### Features
- **Fixed Levels:** Lock characters at any level 1-99
- **Flat Stats:** 4 presets for different challenge difficulties
- **Exp Control:** Disable or multiply experience gain
- **Challenge Modes:** Quick setup for popular challenge runs
- **Safety:** Automatic backups with restore capability

### Use Cases
- Low-level challenge runs (level 1 playthrough)
- Equipment-only scaling (no stat growth from levels)
- Build testing (fixed stats for equipment combinations)
- Speedrun routing (consistent stats across runs)
